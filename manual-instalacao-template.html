Manual de instrução de como abrir o Tema.

Você está adquirindo um modelo completo de um site, feito exclusivamente, para o software de desenvolvimento Blocs App.

Considera-se que já tenha experiência no mínimo básica para utilizar o blocs. Caso possua dúvidas, poderá consultar a documentação Blocs App no botão abaixo:

AJUDA BLOCS APP

Arquivos recebidos:

Ao comprar esses itens você está adquirindo, 2 arquivos principais, confira:

01 – Arquivo Blocs app no formato .blocs que pode ser aberto somente no aplicado de desenvolvimento Blocs.

01 – Arquivo Zip – No aquivo Zip, irá conter o site completo do modelo escolhido no formato HTML, CSS, Java.

OBS, caso venha algum arquivo adicional em formato zip, deve-se extrair o arquivo e instalar eles antes, geralmente são arquivos .bex. Arquivos BEX são complementos utilizado pelo Blocs App, mais conhecido popularmente como plugins.

Para abrir o template Blocs App, basta arrastar o arquivo .blocs que veio nessa compra para cima do atalho do aplicativo que veio nessa compra


Você poderá clicar no arquivo .blocs duas vezes para abrir.


Poderá também clicar com o botão direito e selecionar a opção Abrir.
